# Telnet ChatRoom
## Made by Ben Ayzenberg

## Chat commands
* \all **message**
  * Sends the **message** to all Users connected
* \nick **new nickName**
  * Changes your nickname to **new nickName**
* \dm **__user__ message**
  * Sends the **message** to the **__user__**

## Connect using telnet.
### The default local port is 3000.
